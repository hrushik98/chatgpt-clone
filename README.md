# Getting Started
0. Clone this repository
   ```python
   git clone https://github.com/hrushik98/chatgpt-clone.git
   ```
1. Install Streamlit
   ```python
   pip install streamlit
   ```
2. >cd into the directory where you have cloned this repo
3. run the app
   ```python
   streamlit run app.py
   ```
