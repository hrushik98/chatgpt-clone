# ChatGPT clone

# Getting Started
0. Clone this repository
   ```python
   git clone https://github.com/hrushik98/chatgpt-clone.git
   ```
1. cd into the folder
```python
cd chatgpt-clone
```

2. Install the requirements
   ```python
   pip install -r requirements.txt
   ```

3. run the app
   ```python
   streamlit run app.py
   ```
